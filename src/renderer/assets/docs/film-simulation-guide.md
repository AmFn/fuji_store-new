# 富士胶片模拟参数说明

## 概述

富士相机提供强大的胶片模拟功能，模拟传统胶片的色彩表现。本文详细介绍各参数的含义和使用方法。

---

## 胶片模拟 (Film Simulation)

### 经典胶片模拟模式

| 模式 | 描述 | 适用场景 |
|------|------|----------|
| **PROVIA/标准 (Provia)** | 富士默认标准模式，色彩还原自然，适合风景和人像 | 通用场景 |
| **Velvia/鲜艳 (Velvia)** | 高饱和度、高对比度，色彩非常鲜艳 | 风景、夕阳 |
| **ASTIA/柔和 (Astia)** | 柔和色调，肤色表现优异 | 人像、婚礼 |
| **CLASSIC CHROME** | 低饱和度、高对比，呈现复古电影感 | 街拍、纪实 |
| **PRO Neg. Hi** | 专业负片高调版本，肤色细腻 | 影棚人像 |
| **PRO Neg. Std** | 专业负片标准版本，自然柔和 | 人像 |
| **ETERNA/影院 (Eterna)** | 电影质感，低饱和度、长调曝光 | 视频、电影感 |
| **ETERNA BLEACH BYPASS** | 漂白效果，极低饱和度、高对比 | 艺术创作 |
| **ACROS** | 黑白模式，细腻颗粒感 | 黑白摄影 |
| **单色 (Monochrome)** | 经典黑白，适合街拍 | 黑白摄影 |
| **单色+滤镜 (Monochrome + Ye/O/R/G)** | 带滤镜的黑白，调节反差 | 黑白摄影 |

## 白平衡 (White Balance)

### 自动白平衡 (Auto WB)
- 由相机自动判断场景光源并调整
- 适合快速拍摄

### 预设白平衡
| 预设 | 色温 | 适用场景 |
|------|------|----------|
| 日光 (Daylight) | 5200K | 阳光明媚的户外 |
| 阴天 (Cloudy) | 6000K | 阴天户外 |
| 荧光灯 (Fluorescent) | 4000-5000K | 室内荧光灯 |
| 钨丝灯 (Incandescent) | 3000K | 室内钨丝灯 |
| 闪光灯 (Flash) | 5500K | 使用闪光灯 |

### 白平衡偏移 (WB Shift)
- **R (红)**: +增强红色，-增强青色
- **B (蓝)**: +增强蓝色，-增强黄色
- 范围: -9 到 +9

---

## 动态范围 (Dynamic Range)

### DR100
- 无动态范围扩展
- 适合高调场景（整体明亮的照片）

### DR200
- 扩展动态范围约2档
- 适合中间调为主的场景

### DR400
- 扩展动态范围约4档
- 适合大光比场景（明暗对比强烈）

> **注意**: DR400会降低ISO可用范围

---

## 色调曲线 (Tone Curve)

### 高光 (Highlight)
- 调整高光区域亮度
- 值: -2 到 +4
- **+值**: 高光更亮，细节更多
- **-值**: 高光更暗，压高光

### 阴影 (Shadow)
- 调整阴影区域亮度
- 值: -2 到 +4
- **+值**: 阴影更亮，提亮暗部
- **-值**: 阴影更暗，增加对比

---

## 色彩与锐度

### 色彩饱和度 (Color)
- 调整整体色彩饱和度
- 值: -4 到 +4
- 负值降低饱和度，正值增加饱和度

### 锐度 (Sharpness)
- 调整图像边缘锐度
- 值: -4 到 +4
- **-4到-1**: 柔和，适合人像后期
- **0**: 标准锐度
- **+1到+4**: 增加锐度，风景可用

### 降噪 (Noise Reduction)
- 减少噪点（特别是高ISO时）
- 值: -4 到 +4
- **高数值**: 降噪更强，但可能损失细节

### 清晰度 (Clarity)
- 调整图像中间调的对比度
- 值: -5 到 +5
- 增加微对比度，使图像更有"立体感"

---

## 颗粒效果 (Grain Effect)

### 粗糙度 (Roughness)
- 控制颗粒大小
- 值: 弱 (Weak)、强 (Strong)

### 颗粒大小 (Size)
- 颗粒的粗细程度
- 值: 小 (Small)、大 (Large)

---

## 色彩效果 (Color Chrome)

### 色彩效果 (Color Chrome)
- 增强色彩浓度，特别是红色和黄色
- 值: 关闭 (Off)、弱 (Weak)、强 (Strong)
- 适用于高饱和度场景，避免过曝

### 蓝色色彩效果 (Color Chrome FX Blue)
- 专门增强蓝色表现
- 值: 关闭 (Off)、弱 (Weak)、强 (Strong)
- 适合拍摄天空和水体

---

## 常用配方推荐

### 经典电影感
```
胶片模拟: CLASSIC CHROME
动态范围: DR200
高光: -1
阴影: +2
色彩: -2
清晰度: +2
颗粒效果: 弱/小
```

### 风光摄影
```
胶片模拟: Velvia
动态范围: DR200
高光: -1
阴影: -1
色彩: +2
锐度: +2
```

### 人像摄影
```
胶片模拟: PRO Neg. Std
白平衡: 自动或日光
清晰度: -1
色彩: -1
颗粒效果: 关闭
```

---

## 提示与技巧

1. **包围拍摄**: 同一场景使用不同配方拍摄
2. **RAW+JPEG**: JPEG用配方，RAW保留后期空间
3. **ISO限制**: 使用高DR模式时注意ISO限制
4. **直方图**: 经常检查直方图避免过曝
